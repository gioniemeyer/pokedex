# pokedex

Catch them all is your real test? So, that is yout moment ! (POKEMON!)

## Features

- Sign Up
- Sign In
- List all pokemons
- Catch a pokemon
- Release a pokemon
- Filter user's pokemons

## How to run

1. Clone this repository
2. Clone the back-end repository at https://github.com/gioniemeyer/pokedex-API
3. Follow instructions to run back-end at previously mentioned repository
4. Install dependencies
```bash
npm i
```
5. Run the front-end with
```bash
npm start
```
6. You can optionally build the project running
```bash
npm run build
```
7. Finally access http://localhost:3000 on your favorite browser
